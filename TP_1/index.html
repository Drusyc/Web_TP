Installation de l'environnement

Installer les logiciels : netbeans, firefox et les plugins web developper et firebug.

DOM et CSS

Pour cette page:
  • Explorer le DOM.
  • Tracer l'arbre correspondant en indiquant les types de noeuds (vous pouvez utiliser les outils intégrés à Web Developper)
  • En utilisant le mode CSS de firebug,modifier la CSS pour que les liens sur lesquels passent la souris soient en vert.
  • De même modifier la CSS pour que le menu récapitulatif soit à droite plutôt qu'à gauche, avec la ligne de séparation du bon coté, ajuster les marges et espacements pour que les numéros des items soient après la ligne de séparation.
  • Modifier la couleur de l'élément selectionné dans la barre de menu en jaune.
